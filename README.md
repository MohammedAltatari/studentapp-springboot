# Student Management System

## Description

A Spring Boot CRUD application developed for Programming 3 (CSCI 2308).

The application allows users to manage student records using the following operations:

* Create Student
* Read Students
* Update Student
* Delete Student

## Technologies Used

* Java 17
* Spring Boot
* Spring Data JPA
* Thymeleaf
* MySQL
* Maven

## Student Table

| Field | Type   |
| ----- | ------ |
| id    | Long   |
| name  | String |
| major | String |
| grade | Double |

## Features

* Add new students
* View all students
* Edit student information
* Delete students
* MySQL database integration
